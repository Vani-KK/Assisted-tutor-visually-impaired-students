## AI-Powered Learning Assistant for Visually Impaired Students
This project is an AI-powered wearable learning assistant designed for visually impaired students. 
It uses speech input, RAG-based question answering, and AI tutoring to help students understand study material through audio interaction.

## Features:
- Voice-based question answering
- RAG-based document understanding
- Audio recording interface
- Faculty dashboard for uploading study materials
- Student dashboard for interaction

## Technologies Used:
- React.js
- Node.js
- Firebase Authentication
- OpenAI API
- RAG (Retrieval Augmented Generation)

## Installation:

1. Clone the repository:
git clone "repo-link"

2. Navigate to the project folder:
cd Assisted-tutor-visually-impaired-students

3. Install dependencies:
npm install

4. Start the server:
node server.js

5. Run the frontend:
npm start
